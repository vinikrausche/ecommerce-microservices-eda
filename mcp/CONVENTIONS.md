CONVENÇÕES DO SISTEMA


FRONTEND:
# 1 - Para criar componentes customizados sempre utilize a pasta /src/components, mas não insira nada na pasta ui (shadcn)

# 2 - para chamadas HTTP, utilize api.ts no caminho: /src/services/api.ts

# 3 - componentes devem usar as cores descritas abaixo:
a: Preto carvão (#121212)
→ Fundo, header, footer
Transmite força, elegância, premium.

b: Marrom couro (#6B3E26)
→ Destaques, botões, preço, hover
Conecta direto com couro, botas, artesanal.

c: Bege areia (#D8CFC4)
→ Fundo secundário, cards
Equilibra e não cansa o olho.

d: Off-white (#F5F5F5)
→ Textos em fundo escuro

e:componentes devem ser criados usando arrow function (não use componente classe nem função padrão)

# 4 PAGINAS
a: As páginas ficam na pasta /src/pages/
b:  sempre usando componente função (não arrow function e nem classe)
